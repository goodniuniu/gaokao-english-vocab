## 📝 正文（复制下面整段）

高考英语单词老是背了忘、忘了背，市面上的 App 要么词表固定、要么一打开就广告 😩

于是我打开 **WorkBuddy（混元 Hy3 模型）**，直接甩了一句：
> "帮我写个单文件网页：高考英语单词范围，能随机抽词出题、自动判分、带错题本和历史最佳，打开即用。"

几轮对话后，它真的给了一整份**能直接跑**的 HTML 代码 ✅ 不是一个 Hello World，是块真能用的功能。

它的亮点：
🔹 **会"出题"不是死词表**：内置 1000+ 高考高频词，每次练习随机抽词 + 动态生成干扰项，每次都是新题
🔹 **5 种练习模式**：卡片学习 / 词义选择（看英文选中文）/ 单词选择（看中文选英文）/ 拼写练习 / 智能组卷（随机混刷）
🔹 **错题本**：答错的词自动收集、按词去重、本地保存，还能一键「错题复习」只练错过的新题
🔹 **历史最佳**：各模式最佳正确率跨会话保留，进步看得见 🏅
🔹 **零依赖**：纯 HTML+JS 单文件，双击 `index.html` 就能用，不装环境

最戳我的是——它给的是**真·可运行代码**，还顺手做了正确性自检（随机出题 2400 道、0 错误）。一个 AI 助手的价值，不在于替你写句"你好"，而在于能听懂真实需求、生成真正用得上的东西。

想复刻的宝子：把需求说清楚 → 让 WorkBuddy 出代码 → 本地打开验收，三步搞定。代码我已经开源在 GitHub 啦，链接放评论区 👇
