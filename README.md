# 高考英语单词练习

> 由 **WorkBuddy（混元 Hy3 模型）** 从零生成并逐步完善的高考英语单词练习应用。词库覆盖高考英语词汇（950+ 高频词，含音标 / 词性 / 中文释义 / 英文例句）。

## ✨ 功能

### 核心功能
- 🧠 **智能出题引擎**：每次练习从词库随机抽词 + 动态生成干扰项，每次都是新题
- 📇 **卡片学习**：翻卡片背单词，点击翻转看释义和例句
- 🔤 **词义选择**：看英文选中文（4 选 1）
- 🧩 **单词选择**：看中文选英文（4 选 1）
- ✍️ **拼写练习**：看释义 + 音标，手拼单词（回车提交）
- 🎲 **智能组卷**：SRS 算法混合出题（新词 + 复习词）
- 📖 **复习模式**：只练到期的间隔重复复习词

### 学习增强
- 🔊 **单词发音**：使用浏览器原生 Web Speech API，免费离线朗读
- 📝 **例句展示**：每个单词配有英文例句和中文翻译，提供学习语境
- 🧬 **SRS 间隔重复**：基于 SM-2 算法，科学安排复习时间，最大化记忆效率
- 📊 **掌握度追踪**：新词 → 学习中 → 熟悉 → 已掌握，四级掌握度可视化
- 🎯 **每日目标**：设定每日练习目标，打卡追踪学习进度
- 🔥 **连续天数**：记录连续学习天数（Streak），激励坚持每日打卡

### 用户系统
- 👥 **多用户支持**：本地创建多个学习者档案，数据隔离
- ✍️ **自定义单词**：用户可添加/编辑/删除自己的单词，自动加入练习题库
- 💾 **数据隔离**：每个用户有独立的错题本、SRS 数据、历史记录

### 错题管理
- 📕 **错题本**：答错的词自动收集、按词去重、`localStorage` 本地保存
- 🔍 **错题搜索**：支持按单词或释义搜索
- 🗑️ **逐条删除**：可移除已掌握的错题，也可一键清空

### 体验优化
- 🏅 **历史最佳**：各模式最佳正确率跨会话保留
- 🌙 **暗色模式**：夜间学习友好，支持手动切换
- ⌨️ **键盘快捷键**：A-D 选择答案 / 回车下一题 / 空格翻卡片
- 📱 **响应式设计**：完美适配手机和桌面端

## 🚀 使用

### 方式一：直接打开
双击 `index.html` 用浏览器打开。

### 方式二：本地服务器（推荐）
```bash
cd gaokao-english-vocab
python3 -m http.server 8124
# 浏览器打开 http://localhost:8124/index.html
```

### 快速上手
1. 打开应用后创建一个学习者（输入名字即可）
2. 选择练习模式开始答题
3. 答错的词自动加入错题本
4. 智能组卷模式会根据 SRS 算法自动安排新词和复习

## 🔧 技术栈

- **纯前端**：HTML + CSS + JavaScript，零依赖
- **数据存储**：localStorage（用户数据、SRS 记录、错题本）
- **发音**：Web Speech API (SpeechSynthesis)
- **算法**：SM-2 间隔重复算法
- **兼容**：Chrome / Firefox / Safari / Edge 现代浏览器
