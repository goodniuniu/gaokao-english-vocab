# 高考英语单词练习生成器

> 由 **WorkBuddy（混元 Hy3 模型）** 从零生成的单文件网页应用。单词范围：高考英语词汇（内置约 1000+ 高频词，含音标 / 词性 / 中文释义）。

## ✨ 功能

- 🧠 **题目生成引擎**：每次练习都从词库**随机抽词 + 动态生成干扰项**，不是死题库——每次都是新题。
- 📇 **卡片学习**：翻卡片背单词，点一下看释义。
- 🔤 **词义选择**：看英文选中文。
- 🧩 **单词选择**：看中文选英文。
- ✍️ **拼写练习**：看释义 + 音标，手拼单词（回车提交）。
- 🎲 **智能组卷**：随机 10 题混刷（三种题型混合）。
- 📕 **错题本**：答错的词自动收集、按单词去重、`localStorage` 本地保存，可回看释义 / 一键清空；「错题复习」模式只练错过的新题。
- 🏅 **历史最佳**：各模式最佳正确率跨会话保留。
- 📊 **学习统计**：词库量 / 错题数 / 已练题数 / 最佳正确率，首页一目了然。
- 📱 **零依赖**：纯 HTML + JS 单文件，双击 `index.html` 即用，数据存本地浏览器，刷新不丢失。

## 🚀 使用

方式一：直接双击 `index.html` 用浏览器打开。
方式二（推荐，避免本地文件限制）：起一个静态服务

```bash
cd gaokao-english-vocab
python3 -m http.server 8124
# 浏览器打开 http://localhost:8124/index.html
```
